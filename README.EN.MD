## рҹҺӨ LRC Lyrics

Musicpy Downloader supports **three lyrics modes**:

- вң… **Embedded in the file** вҖ” The lyrics travel along with the song, synchronized to the millisecond
- вң… **Link only** вҖ” Download without lyrics if you don't need them
- вң… **Separate `.lrc` file** вҖ” Save the lyrics as an external file

```
[00:12.00] рҹҺө The lyrics appear synchronized...
[00:15.30] рҹҺө while you listen to the song.

[00:18.60] рҹҺө No external apps. No configuration.

```
