## ✨ Что такое Musicpy Downloader?

**Musicpy Downloader** — это инструмент с интерфейсом на Python, который позволяет скачивать музыку с самых популярных платформ мира — с поддержкой встроенных текстов песен, множеством форматов и полной визуальной настройкой.

> 🎯 Один инструмент. Пять платформ. Никаких ограничений. Никакой регистрации.

---

## 🚀 Основные возможности

| Возможность | Описание |
|---|---|
| 🎨 **Настраиваемые цвета** | Измените цвета интерфейса по своему вкусу |
| 📁 **7 форматов загрузки** | MP3 (320k и 128k), M4A (AAC), FLAC, WAV, OGG, OPUS, AAC |
| 🌐 **7 источников загрузки** | Несколько бэкендов для максимальной доступности |
| 🎤 **Встроенные тексты LRC** | Синхронизированы и встроены прямо в файл |
| 🔗 **Загрузка только по ссылке** | Вставьте ссылку — и готово, никаких лишних настроек |
| 📋 **Поддержка плейлистов** | Загружайте целые плейлисты за один раз |
| 🎵 **5 платформ** | YouTube, Spotify, SoundCloud, Deezer и Tidal |
| 📁 **Смена папки загрузки** | Выберите, куда сохранять музыку |

## 🎤 Тексты песен LRC

Musicpy Downloader поддерживает **три режима работы с текстами**:

- ✅ **Встроен в файл** — Текст сохраняется вместе с песней, синхронизирован до миллисекунды
- ✅ **Только ссылка** — Загрузка без текста, если он вам не нужен
- ✅ **Отдельный файл `.lrc`** — Сохранение текста как внешнего файла

```
[00:12.00] 🎵 Текст появляется синхронизированно...
[00:15.30] 🎵 пока вы слушаете песню.
[00:18.60] 🎵 Без сторонних приложений. Без настроек.
```

## ⚙️ Установка

1. Скачайте `.zip` файл из папки релизов
2. Запустите `install.py`
3. После прохождения 6 проверок требований
4. Запустите `run.bat`

---

> **⚠️ ВАЖНО:** Файлы `ffmpeg.exe` и `ffprobe.exe` будут включены в загрузку на случай ошибок при установке через `install.py`.

> **⚠️ ВАЖНО:** Окно командной строки, которое открывается, **не следует закрывать вручную** — оно закроется автоматически после завершения работы программы.
