## ✨ 什么是 Musicpy Downloader？

Musicpy Downloader 是一款基于 Python 接口的工具，可让您从全球最流行的音乐平台下载音乐——支持嵌入式歌词、多种格式以及完整的视觉自定义功能。

> 🎯 一个工具，五大平台，零下载限制。无需注册。

## 🎤 LRC 歌词

Musicpy 下载器支持**三种歌词模式**：

- ✅ **嵌入文件** — 歌词与歌曲同步，精确到毫秒

- ✅ **仅链接** — 如果您不需要歌词，可以选择不下载

- ✅ **单独的 `.lrc` 文件** — 将歌词保存为外部文件

```

[00:12.00] 🎵 歌词同步显示...

[00:15.30] 🎵 在您聆听歌曲时。

[00:18.60] 🎵 无需外部应用，无需配置。

```

## ⚙️ 安装

1. 从发布文件夹下载 .zip 文件

2. 运行 install.py

3. 满足 6 项要求后

4. 运行 run.bat

---

#*重要提示* 如果 install.py 出现错误，我们将提供 ffmpeg.exe 和 ffprobe.exe 供您下载。

重要提示：请勿删除打开的命令提示符窗口；程序关闭后，该窗口将自动关闭。
