## рҹҺӨ Letras LRC

Musicpy Downloader soporta **tres modos de letra**:

- вң… **Incrustada en el archivo** вҖ” La letra viaja junto con la canciГіn, sincronizada al milisegundo
- вң… **Solo el link** вҖ” Descarga sin letra si no la necesitas
- вң… **Archivo `.lrc` separado** вҖ” Guarda la letra como archivo externo

```
[00:12.00] рҹҺө La letra aparece sincronizada...
[00:15.30] рҹҺө mientras escuchas la canciГіn.
[00:18.60] рҹҺө Sin apps externas. Sin configuraciГіn.
```
